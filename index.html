Origem do HTML

HTML: Significa Hyper Text Markup Language ou traduzindo para o português Linguagem de Marcação de HiperTexto. Foi desenvolvida por Tim Berners-Lee no ano de 1991.

Ao contrário do que muita gente pensa, o HTML não é uma linguagem de programação, como JavaScript, Python, PHP, entre outras linguagens que são do Servers Side, ele é, como o próprio nome diz uma Linguagem de Marcação, ou seja, uma pessoa que utiliza o HTML não programa com HTML, mas sim Desenvolve com HTML, ele esta no Clients Side.

O que são Clients?

Clients são as pessoas que consomem a informação da internet por meio de algum dispositivo, essas informações são encaminhadas dos servidores ou serves que entre outras coisas hospedam os sites que são acessados. Os serves ao enviar os arquivos de HTML, CSS, JS, imagens e áudio, entre outros para para navegador via cache, assim dessa forma o navegador recodifica os arquivos e exibe na tela do dispositivo.

Os principais dispositivos utilizados atualmente pelos usuários são:

  • Smartphones;
  • Notebooks;
  • Tablets;
  • Smart Tvs.

O que são Servers?

Os Servers, ou servidores podem ser locais onde uma rede de computadores conectados acessam para ter acesso a arquivos e informações ou servidores de internet, onde estão hospedados as informações que os Clients terão acesso. Os servidores podem ser dos tipos de arquivo, segurança (Firewall), Streaming, E-mail e Web.

O que são Tags e as principais utilizadas para quem esta começando HTML?

As Tags HTML são etiquetas ou marcadores que indicam ao navegador como interpretar e exibir o conteúdo de uma página da web. Elas são usadas para definir a estrutura e o layout da página, bem como para adicionar e formatar o texto, imagens, links e outros elementos que compõem a página

Para quem esta começando a aprender essa linguagem Web, as principais Tags utilizadas são:

  • h1 a h6 - Define os cabeçalhos da página, sendo o h1 o mais importante e o h6 o menos importante.
  • p - Define um parágrafo de texto.
  • a - Define um link para outra página ou recurso na web
  • img - Define uma imagem a ser exibida na página.
  • ul, ol, li - Define uma lista não ordenada ul, uma lista ordenada ol e um item de lista li
  • form, input, button, select - Define um formulário form, um campo de entrada de dados input, um botão button e uma lista de opções select
